Hi, I’m **Amy**, a digital and traditional artist based in the UK.

## About me

I love:

- **Digital illustration** – characters, environments, and stylised portraits  
- **Traditional media** – ink, markers, and mixed media sketchbook work  
- **Storytelling** – using visuals to build worlds and moods

When I’m not drawing, I’m probably:

- Sketching ideas in a notebook  
- Learning new techniques  
- Playing with colour palettes
